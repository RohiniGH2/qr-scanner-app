Complete User Guide & Instructions
Step 1: Student Registration
  • Click "Add Student" button
  • Fill in student details (name, roll number, class)
  • Upload student photo (optional but recommended)
  • Submit to generate unique QR code automatically
Tip: Use clear, recent photos for better identification
Step 2: QR Code Management
  • Each student gets a unique QR code
  • Click QR code image to view in modal
  • Download individual or all QR codes
  • Print QR codes for physical distribution
Tip: QR codes link directly to student profiles
Step 3: Access & Scanning
  • Share server URL with network devices
  • Scan QR codes using phone camera
  • View detailed student information instantly
  • Access from any device on same network
Tip: Works on phones, tablets, and computers

Network Setup Guide
1
Ensure Same Network: All devices must be connected to the same WiFi or LAN network as this server.
2
Share URL: Copy the server URL ({{ local_ip }}:8080) and share with other devices.
3
Test Connection: Use the "Network Diagnostics" feature to verify connectivity.
4
Scan QR Codes: Use any camera app to scan QR codes and access student profiles.
Troubleshooting Common Issues
Cannot Access from Mobile
  • Check if devices are on same network
  • Try disabling firewall temporarily
  • Use "Network Diagnostics" tool
  • Ensure server is running on port 8080
QR Code Not Working
  • Ensure good lighting when scanning
  • Hold camera steady and at proper distance
  • Try different QR scanner apps
  • Check if URL in QR code is correct
Photo Upload Issues
  • Use JPG, PNG, or GIF formats
  • Keep file size under 5MB
  • Ensure clear, well-lit photos
  • Try refreshing if upload fails
